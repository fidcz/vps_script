# vps_script
自用的脚本

## trojan_alpine.sh
> 快速安装trojan

```
wget -O trojan_install.sh https://raw.githubusercontent.com/fidcz/vps_script/refs/heads/main/trojan_install.sh && chmod +x trojan_alpine.sh && ./trojan_install.sh
```
或
```
curl -o trojan_install.sh https://raw.githubusercontent.com/fidcz/vps_script/refs/heads/main/trojan_install.sh && chmod +x trojan_alpine.sh && ./trojan_install.sh
```
