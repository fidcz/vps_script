## trojan_alpine.sh
> 快速安装trojan

> wget -O trojan_alpine.sh https://raw.githubusercontent.com/fidcz/vps_script/refs/heads/main/trojan_alpine.sh && chmod +x trojan_alpine.sh && ./trojan_alpine.sh

> 或

> curl -o trojan_alpine.sh https://raw.githubusercontent.com/fidcz/vps_script/refs/heads/main/trojan_alpine.sh && chmod +x trojan_alpine.sh && ./trojan_alpine.sh
